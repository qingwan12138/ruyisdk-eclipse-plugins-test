# 开发板选择框排序

## 操作步骤
1.点击 RuyiSDK -> Package Explorer -> current device(none selected)
2.选择对应的开发板

## 预期结果
可以按照 name、ID、Variants进行选择顺序或者逆序排列

## 测试结果
可以按照 name、ID进行选择顺序或者逆序排列，但是Variants字节顺序以及逆序排列仍混乱
